# Contact Book App (CSV Storage)

This is a console-based contact book app in Python that stores contacts in a CSV file.

## Features

- Add, view, update, delete contacts
- Data is saved to a CSV file
- Test cases included

## Steps to Run in PyCharm IDE

1. Open PyCharm.
2. Open the `ContactBookCSVApp` directory.
3. Run `contact_book_csv.py` to use the app.
4. Run `test_contact_book_csv.py` to test functionality.

## Real-World Test Scenarios

- Add new contact and check CSV
- Prevent duplicate names
- Update and verify saved values
- Delete contact and check removal
- Read all data from CSV
